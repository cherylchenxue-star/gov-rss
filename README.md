# 政府政策聚合 RSS

聚合多个政府部门政策信息的 RSS 订阅源。

## 访问地址

- **网页预览**: `https://cherylchenxue-star.github.io/gov-rss/`
- **RSS 订阅**: `https://cherylchenxue-star.github.io/gov-rss/rss.xml`

## 数据来源

- 工信部信息通信发展司
- 国家数据局
- 国家发改委
- 国家网信办

## 更新频率

每天 3 次（北京时间 06:00、14:00、22:00）

## 功能特性

- ✅ 自动抓取政策原文摘要
- ✅ 多源聚合去重
- ✅ 反爬对策（随机延时、User-Agent轮换）
- ✅ RSS 2.0 标准格式
